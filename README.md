# Simple Calculator (Expo React Native)

Ocean Professional themed calculator with basic arithmetic.

- Start web preview: npm run web
- Start Expo dev: npm start

Environment variables (optional):
- EXPO_PUBLIC_LOG_LEVEL=verbose   # extra debug logs
- EXPO_PUBLIC_HEALTHCHECK_PATH=/health
- EXPO_PUBLIC_FEATURE_FLAGS=debug
- EXPO_PUBLIC_EXPERIMENTS_ENABLED=true

Features:
- Two-line display (expression + current value)
- 4x5 keypad with 0 spanning two columns
- Operators row and equals accent
- divide-by-zero -> Error state; AC to reset
